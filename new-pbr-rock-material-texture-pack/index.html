New PBR rock material texture pack

This texture set was created as a series of studies on different rocky materials. The whole series includes the following rock materials:

  • Dolomite
  • Sandstone
  • Slate
  • Tuff
  • Dry stone (made out of limestone)

Technical info:

  • 4k EXR 16 bit files
  • normal maps are for openGL software (Blender, etc.). Flip the green channel to use them with DirectX software
  • using the height maps is necessary to obtain the correct result.

The material was created using Substance Designer. The material preview image has been rendered using Blender 3.0. The license is CC0. For more details about the license, click here.

The material is available for free. I would gladly accept your support, though. If these textures were of any use to you and you want to, feel free to pay what you want for them.
